# PROJECT: Odontotos Railway Landing Page

## Goal
Create a cinematic tourism landing page for the historic Odontotos rack railway in Diakopto Greece.

The railway crosses the Vouraikos Gorge UNESCO Global Geopark.

## Site Philosophy
Experience page, not a full tourist guide.

## Structure

1. **Hero**
2. **Experience**
3. **UNESCO Gorge**
4. **Railway History**
5. **Diakopto Starting Point**
6. **Route Map**
7. **Visitor Info**
8. **CTA Discover Diakopto**

## Design Style

- National Geographic
- Cinematic photography
- Large spacing
- Magazine layout
